# Programme-Financial-Health-Dashboard-
## Project Overview

This project presents a Programme Financial Health Check Dashboard that provides a comprehensive financial and delivery performance overview of a multi-project programme portfolio for the year 2025.
The dashboard is designed to support programme managers, finance officers, and senior leadership in monitoring delivery progress, assessing forecast accuracy, and identifying financial and implementation risks across projects. It integrates forecasted budgets, actual delivery, burn rates, and cash positions to enable evidence-based decision-making and proactive programme management.
